openBD優先、Google Books fallback、手動入力継続を前提。 |  |  |  |  |  |  | 
連携先 | URL | 用途 | 注意点 |  |  |  | 
openBD | https://openbd.jp/ | 日本語書誌・書影が強い。/get?isbn=... を提供。 | 利用規約上、書誌・書影等は本の販促/紹介目的に限る。キャッシュ反映にも注意。 |  |  |  |

Priority-2 | Google Books | openBD未取得時の fallback。q=isbn: を使用。 |  |  |  |  |
